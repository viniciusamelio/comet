# comet-docs

Documentation site for the `comet` crate, built with [Fumadocs](https://fumadocs.dev) on Next.js.

## Develop

```sh
npm install
npm run dev
```

Opens at `http://localhost:3000`. Content lives in `content/docs/*.mdx`, ordered by `content/docs/meta.json`.

## Build

```sh
npm run build
npm run start
```
